# SiapGuru v1.0.0

Rilis awal SiapGuru untuk workflow `wali_kelas` dan `guru_mapel`.

## Perubahan utama

- Aktivasi lisensi sudah memakai Firebase.
- Mode `wali_kelas` dan `guru_mapel` memakai workspace lokal terpisah.
- Export dan template diarahkan ke `Documents\SiapGuru`.
- Smart Ketuntasan sudah memakai target rentang nilai akhir.
- Buat Raport menghasilkan file Excel dengan satu sheet per siswa.
- Aplikasi dibuka dalam keadaan maximized.

## Artefak rilis

- `SiapGuru.exe`
- `Setup-SiapGuru-1.0.0.exe`

## Catatan distribusi

- Untuk user umum, bagikan `Setup-SiapGuru-1.0.0.exe`.
- `SiapGuru.exe` dipakai jika ingin distribusi portable/internal.
