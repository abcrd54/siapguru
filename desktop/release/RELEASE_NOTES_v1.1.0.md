# SiapGuru v1.1.0

- Menambahkan menu `Modul` untuk upload PDF pembelajaran per kelas dan mapel.
- Menambahkan menu `Soal` untuk generate soal otomatis dari teks modul.
- Integrasi desktop ke backend admin untuk upload modul dan generate soal.
- Hasil generate soal sekarang disimpan dalam format JSON dan bisa dilihat dari aksi tabel.
- Menambahkan progress dialog bertahap saat upload modul dan generate soal.
- Menambahkan pengaturan lisensi berbasis fitur untuk mengatur menu yang tampil di desktop.
- Menambahkan opsi hapus modul dan penghapusan file cloud saat konfigurasi Cloudinary tersedia.
